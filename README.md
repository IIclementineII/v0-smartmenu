# SmartMenu Agent — Frontend 🍽️

> AI-powered menu experience for Jade Palace Restaurant.

This is the frontend for [SmartMenu Agent](https://github.com/IIclementineII/smartmenu-agent), built for the [Google Cloud Rapid Agent Hackathon 2026](https://rapid-agent.devpost.com/) — MongoDB Partner Track.

**Live Demo:** https://v0-smartmenu.vercel.app/  
**Backend Repository:** https://github.com/IIclementineII/smartmenu-agent

---

## Pages

**Customer View** — Chat with Jade, our AI concierge. Filter dishes by dietary needs, ask about allergens, get personalized recommendations.

**Owner Dashboard** — Manage menu items, update prices and stock, toggle availability — all via natural language AI commands.

---

## Tech Stack

Next.js · Tailwind CSS · Deployed on Vercel

---

## Local Development

```bash
git clone https://github.com/IIclementineII/v0-smartmenu.git
cd v0-smartmenu
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
